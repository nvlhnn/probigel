# Probiogel Website

Probiogel - Antiseptik Modern berbasis HOCl. 
This is the official landing page and product website for Probiogel, built as a fast, static website using Eleventy (11ty) and styled with Tailwind CSS v4.

## 🚀 Tech Stack

*   **Static Site Generator:** [Eleventy (11ty) v3](https://www.11ty.dev/)
*   **Templating Engine:** [Nunjucks](https://mozilla.github.io/nunjucks/) & HTML
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Content Management:** [Decap CMS](https://decapcms.org/) (configurable in `src/admin/`)
*   **Deployment:** [Netlify](https://www.netlify.com/)

## 📂 Project Structure

```text
├── .eleventy.js       # Eleventy configuration
├── netlify.toml       # Netlify deployment configuration
├── package.json       # Project dependencies and npm scripts
├── src/               # Main source code
│   ├── _data/         # Global singleton data (site settings, form options)
│   ├── _includes/     # Layout templates and partials (header, footer, etc.)
│   ├── admin/         # Decap CMS configuration (config.yml)
│   ├── content/       # CMS-managed content collections
│   │   ├── articles/      # Article markdown files and collection defaults
│   │   ├── faq/           # FAQ markdown files and collection defaults
│   │   ├── products/      # Product JSON entries
│   │   ├── testimonials/  # Testimonial JSON entries
│   │   └── wounds/        # Wound type JSON entries
│   ├── css/           # CSS source files (input.css for Tailwind)
│   ├── images/        # Static image assets
│   ├── js/            # Client-side JavaScript
│   └── *.html / *.md  # Pages (index.html, produk.html, tentang.html, dll.)
└── _site/             # Compiled static output (generated by Eleventy, do not edit manually)
```

## 🛠️ Local Development

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (LTS version recommended).

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
To start local development with live-reloading:
```bash
npm run dev
```
This single command runs both Eleventy's local server and the Tailwind CSS CLI in watch mode simultaneously.
* The website will be available at `http://localhost:3000`

### Other Scripts
*   `npm run build`: Builds the production-ready site. It compiles HTML through 11ty and minifies the Tailwind CSS. The output is placed in the `_site/` directory.
*   `npm run dev:11ty`: Runs only the Eleventy development server.
*   `npm run dev:css`: Runs only the Tailwind CSS watcher.
*   `npm run clean`: Deletes the `_site/` output folder.

## 📦 Deployment

This project is configured to be deployed on **Netlify**.
The `netlify.toml` file contains the build settings:
*   **Build Command:** `npm run build`
*   **Publish Directory:** `_site`

## 🎨 Styling with Tailwind CSS v4

This project uses the latest Tailwind CSS v4 via `@tailwindcss/cli`. 
*   The main entry point for CSS is `src/css/input.css`.
*   The compiled output is sent directly to `_site/css/style.css`.
*   Eleventy configuration (`.eleventy.js`) specifically does *not* passthrough copy the `src/css` directory, allowing Tailwind to handle the build process completely.
